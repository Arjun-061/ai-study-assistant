# ai-study-assistant
# 📚 AI Study Assistant using RAG and Endee

## 🚀 Overview
This project implements a Retrieval-Augmented Generation (RAG) based AI study assistant using vector embeddings and Endee.

## 🎯 Features
- Load study materials
- Convert text into embeddings
- Store embeddings in vector database (Endee)
- Retrieve relevant content based on query
- Generate answers using retrieved context

## 🧠 Workflow
User Query → Embedding → Search in Endee → Retrieve Top Matches → Generate Answer

## ⚙️ Tech Stack
- Python
- Sentence Transformers
- Endee (Vector Database Concept)

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
